## Building Blocks

<CardGrid stagger>
  <Card title="Index" icon="seti:notebook">
    A library with implementations for `IndexReader` and various `IndexWriter`
    strategies, along with `IndexStore` implementations.
  </Card>
  <Card title="Pack" icon="seti:zip">
    A library for writing data into packs (`PackWriter`) and reading them back
    (`PackReader`), facilitating the creation and retrieval of
    content-addressable data. While writing data into packs, Index Records can
    be created when provided an `IndexWriter` implementation.
  </Card>
  <Card title="Streamer" icon="seti:video">
    A library for streaming verifiable data associated with a requested
    multihash, based on available index records and stored packs.
  </Card>
</CardGrid>
